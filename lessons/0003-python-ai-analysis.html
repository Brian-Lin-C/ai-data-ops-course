2. AI 辅助分析 — 新范式

来源：Wes McKinney（pandas 创始人，2026 年播客）：「Python 是为人类设计的。AI 刚刚改变了一切。」

三种 AI 辅助模式

① AI 写，人审查 — 适合标准 ETL、简单可视化。你用自然语言描述需求，AI 生成代码，你检查逻辑正确性。

② 人写框架，AI 填细节 — 适合复杂分析。你搭好分析框架和关键逻辑，AI 补完具体实现。

③ NL2SQL / NL2Pandas — 适合快速探索。用自然语言直接查询数据："上月各渠道 GMV 环比变化" → AI 生成 SQL 或 pandas 代码。

LlamaIndex PandasQueryEngine（来源：LlamaIndex 官方文档）

# 自然语言直接查 DataFrame
from llama_index.experimental.query_engine import PandasQueryEngine

query_engine = PandasQueryEngine(df=df)
response = query_engine.query(
    "各品类上月销售额排名，以及环比增长最快的品类"
)
print(response)

⚠️ AI 辅助分析的铁律：永远验证 AI 生成的代码——跑一遍、看结果、用业务逻辑判断。AI 不知道你的数据有没有缺失值、列名对不对。

3. 完整分析流程模板

# 标准数据分析工作流
# Step 1: 数据加载 + 初步探索
df = pd.read_csv('data.csv')
print(df.info(), df.describe(), df.isnull().sum())

# Step 2: 数据清洗（可让 AI 辅助）
df['date'] = pd.to_datetime(df['date'])
df = df.dropna(subset=['amount'])

# Step 3: 特征工程
df['month'] = df['date'].dt.month
df['is_weekend'] = df['date'].dt.weekday >= 5

# Step 4: 分组分析
result = df.groupby(['month','channel']).agg({'amount':'sum','user_id':'nunique'})

# Step 5: 可视化
import matplotlib.pyplot as plt
result['amount'].unstack().plot(kind='bar', stacked=True)
plt.title('Monthly GMV by Channel')
plt.savefig('gmv_analysis.png')
