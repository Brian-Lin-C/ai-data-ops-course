# AI数据运营策略岗 — 7课求职准备课程

> 针对「AI+数据运营策略」类岗位的系统化学习路径。覆盖 JD 三大核心：数据分析 + 大模型应用 + 项目管理。

## 课程信息

| 项目 | 内容 |
|------|------|
| 课时 | 7 课 |
| 周期 | 建议 15 天（每天 2-3h） |
| 前置知识 | 基础 SQL + Python（数据科学硕士水平即可） |
| 目标岗位 | AI数据运营 / 数据策略 / 业务数据分析 |
| 适用地区 | 广深为主，京沪亦可 |

## 为什么这个课与纯数据分析课不同？

- 📌 每个模块都融入 **AI 工具链**（LLM辅助SQL、自动报告、NL2SQL）
- 📌 第5课「大模型赋能」是核心差异化——这是 JD 最特别的要求
- 📌 第6课「数据项目管理」覆盖 JD 第3项职责，纯DA课程不教这个
- 📌 第7课融合真实面试题（Airbnb/OpenAI 面试指南 + 国内面经）

## 学习路径

```
第❶阶段 (1-4周): 打地基
  → Lesson 1-2: SQL + 业务指标体系

第❷阶段 (5-8周): 上工具 + 建差异化
  → Lesson 3-5: Python/AI + BI + 大模型应用

第❸阶段 (9-12周): 补PM + 投简历
  → Lesson 6-7: 项目管理 + 面试冲刺
```
